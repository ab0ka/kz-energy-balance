\chapter{Interactive Dashboard for Self-Exploration}
\label{app:dashboard}

All analytical visualizations presented in Chapter~\ref{ch:energy_balance}
(Figures~\ref{fig:plant_overview}, \ref{fig:trade_dashboard},
\ref{fig:res_potential}, and~\ref{fig:decom_outlook}) are also available
as a single \textbf{interactive Power BI Desktop file},
included in the project repository:

\begin{center}
    \texttt{Article/Final/Energy.pbix}
\end{center}

The interactive version supports:
\begin{itemize}
    \item Hover-tooltips with detailed plant-level data
        (commissioning year, technology, owner);
    \item Cross-filtering between visuals
        (e.g.\ click on a fuel type in the donut chart to filter the map);
    \item Year-range slicing for historical exploration;
    \item Drill-through to underlying data tables.
\end{itemize}

To open the file, install Microsoft Power BI Desktop (free, available on
Windows from the official Microsoft Power BI website) and double-click the
\texttt{.pbix} file. No additional setup is required --- all data is
embedded in the file.
